## 1. Clone the repository
```bash
git clone <repo-link>
```

```bash
cd Flight-Schedule-Management
```

```bash
npm install
```
## 3. Run the development server
```bash 
npm run dev
```

[http://localhost:5173](http://localhost:5173)

Material UI (MUI)
